Basics of css

Published:No idea

You know the basics of html. But you want to add some styling. Well then you have come to the right place! Here you will learn how to customize buttons,text, images, and much more. I do have a few things to suggest before we start:

  1. Know your html. I suggest you do my basics of html and use that as your html for this post.
  2. Know what a button is. We are going to be customizing buttons here so you may want to think about that.
  3. Put down in your html file all the header tags. Im serious about this.

Now lets get into this.





First things first

Making the css file.

First you will want to make your css file next to your html file. And you want to name it style.css And of course make sure you have your file name extensions shown. And click yes if windows asks you about changing the file name extension. Like this:

Of course it both the files are empty so thats okay.



Linking the files

Now one thing to know is that you cant just open a css file. You need to open it in a code editor. You dont need to open it in a browser.

Now to link a html file to a css file, you need to make sure the css file is in the same folder as the html file or you know the path to the html and css files. Then below your meta tags write this:<link rel="stylesheet" href="style.css">

And thats that.

Testing the link.

Now in your css write


body{
    background-color: red;
}

And it shoukd come up like this:

And then you have a sucsessful link!

Antomony

The {} are called selectors or rulesets. The words before the selectors are the things you control. For example If I did not want a red background color then I can make it green. Or i could change the text so it has a driffent font. Now lets dig into more of the body features!





Background colors and more!

In the body ussaly you control the basic things. Here are some of those basic things:

  1. Font family
  2. Text align
  3. Background weather it will be a solid color or gradient or even a split color page!

Now lets say we want to make the text in the center. Then under you background color write this:text-align:center;And it should come up like this:

And now you see the text is in the center!





Text colors

Now lets say you want to make the h1 text green color. then make and h1 selector like this: 


h1{
    color:green;

h1{
    color:green;
    font-size:75px;
}

Margin

As a begginerr you dont have to read this section but I suggest it. Bassicly the margin adds space. So lets say I want to make my margin 10px for the h3. Then lets make the text align for the h3 left and put your margin of 10px like this: 


h3 {
    text-align:left;
    margin:10px;
}

And it turns out like this:

You can see how the margin has added some space between the left. Ten pixels to be precise.

It is a very handy thing, because sometimes in css you want your text to go to the left but you dont want it to go to the very left.
