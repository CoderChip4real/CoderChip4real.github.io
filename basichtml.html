What is html?

Html stands for

Hyper
Text
Markup
Language Its a way of displaying information on your browser. Now renember this is not a programming language. Its a markup language. Html may not be able to control content like Javascript, but its still extremly important.





Making and opening the html file



Creating the file

To make the file this is how you want to do it:

  1. Open file explorer and choose where you want to put the file

  3. Then open it in your browser
  4. Then it will look blank. Thats okay since the file does not have anything in it.


Now one thing to know is that its a file on your computer, so nobody else can see this file.

Writing the code

Html tags

Html is made up of tags. Tags are special text that you use to markup or distingush your parts of a webpage. Hence the Hyper text markup language. Now the first thing we want to do in our html file is at the top put is <!DOCTYPE html> Then after that we want to type in these tags:<html></html> Seems a bit weird because we have 2 tags that say html in it, but the <!DOCTYPE html> tag declears that this document is html. The <html></html> tags decleare that everything inside these 2 tags are html. Now in the gap between <html></html> we want to make a space bar like this: 


<!DOCTYPE html>
<html>

</html>


Simple.



Next in the gap of the <html> </html> we want to put this in:this is my <b> website </b>

Now when we reload the browser it should look like this:

you will see the website word written in bold. that is what the bold tag does or <b> </b> And now you just wrote code!feel happy yet?

The inside of a html tag



As you know html is made up of tags. A <> represents the start of a tag. A </> Shows the end of a tag. But some tags also dont require the </> I will get into 1 or 2 later.





Text tags.

Now lets sat we want to make that text bigger. we would do that with the header tags or h tags. Get rid of the this is my website, and write in your code: 


<!DOCTYPE html>
<html>
        <h1>hello</h1>
        <h2>hello</h2>
        <h3>hello</h3>
        <h4>hello</h4>
        <h5>hello</h5>
        <h6>hello</h6>
</html>
And it should look like this:

Notice something? Thats right! h1 is the biggest and h6 is the smallest! We dont use h4, h5, or h6 that much though. Now we are going to learn about breaks, or <br>





Breaks

This is one of the few tags that dont have a closing tag. Delete the h 3,4,5 tags and then in between the h1 and h6 tags do this: 


 <!DOCTYPE html>
<html>
        <h1>hello</h1>
        <br>
        <br>
        <br>
        <h6>hello</h6>
</html>

Then it will look like this:





body and head tag!

Now copy all the code inside you html tags, delete it then inside the tags of html then make a tag like this called body and give it an end body tag. 


<!DOCTYPE html>
<html>
    <body>
        <h1>hello</h1>
        <br>
        <br>
        <br>
        <h6>hello</h6>   
    </body>
</html>

Then make a tags called <head> </head> And place them out side of the body but, put them in the html tag above the body like this:


<!DOCTYPE html>
<html>
    <head>
    </head>
    <body>
        <h1>hello</h1>
        <br>
        <br>
        <br>
        <h6>hello</h6>   
    </body>
</html>

And in the head make these 3 tags: 


<head>
    <meta charset="uft-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

Now these meta tags let the website be responsive, which meens it can display properly on all devices,lets our symbols and letters go into bytes which are used by computers. Now we need the title tag. This will be the title of the website. type in the head after the meta tags this: 


<title>My website</title>

Links

Anchor tag

Links are a common way we get around the internet. To do this we use the <a></a> tag or anchor tag.

Making a anchor tag

First in your code type your anchor tag and then inside the opening tag write href="" like this: 


<a href=""></a>

And then inside the"" type the link you want. You will also want to include the full link. so if you wanted to make a link to google it would be like this: 


<a href="https://www.google.com/"></a>

And then in between the <a></a> tags, write google. Then it should come up like this:


<a href="https://www.google.com/">Google</a>
