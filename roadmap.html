Coding languages

With coding, if you have never done it before you might have thought 'Oh well coding is hard but its just one thing. Not a whole lot of other things.'Sorry to say but you are either 1/4 right or 1/2 right. In coding there is more than one type of code. The first thing everyone needs to learn about code is to know your coding languages. The most common coding languages are: Html, css, And javascript. Alright I know there are more coding languages but these are the ones I mostly use and these 3 I use all the time. Html and css are really the stuff that is used to make websites. Like this one right now.



Html

The first language you should probably learn is Html. Stands for Hyper Text Markup Language Its like the base of the website. It lets you make the text, lists and paragraphs and lets you make titles. You can also do this with javascript as well, but I like to use javascript to make games. Thats my suggestion though.



Css

The second language to learn is Css Which stands for Cassading Style Sheets. It adds the color font, makes the webpage pretty, and can customize pretty much every single bit of html. Buttons,text,Paragraphs. You name it.



Javascript

What I describe Javascript is like adding interactivicty. I have used it in the past to help design webpages but I dont really use it with webpages unless I am making shapes and games. Still its essential. A good place to learn it is at Grashopper





Code editors



What is a code editor?

A code editor is what we use to make code. Its also used to make files for html, css, and javascript. And depending on what sort of code editor you have, you might get special extensions and stuff like that.



What code editor should I use?

There are multiple code editors out there but here are some good free ones

  • Visual Studio Code
  • Sublime text
  • Notepad ++
  • Brackets

Which one do I use you may be asking? I use Visual studio code because you can customize your theme, its open sourced, easy to use and it can give definitions of html tags! I also use it because it can come up with suggestions and if you click on a suggestion it will automaticly type it for you!(Unless you are writing html tags you have to add >)
